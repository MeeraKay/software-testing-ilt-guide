# Module 1: Instructor-Led Training

This module is the core of the day: you'll get oriented in Test Manager, see how Agentic Testing fits into the picture, then move into Studio to build, run, and manage automated tests end to end.

## What you'll do

- Import a real project into Test Manager and learn the typical testing flow
- See how Autopilot for Testers and custom AI agents extend a tester's day-to-day work
- Build an automated test case in Studio, using the Object Repository for reusable UI elements
- Connect Studio to Test Manager and turn a manual test case into an automated one
- Apply data-driven testing so one test case can cover many scenarios
- Configure access, licensing, and execution resources in Orchestrator
- Publish test cases to Orchestrator and execute them — manually and automatically — from Test Manager
- Build your own AI agent
