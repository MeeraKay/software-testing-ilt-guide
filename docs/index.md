# Welcome to the 1 Day Software Testing ILT

**Thank you for joining the UiPath Software & Agentic Testing Instructor-Led Training!**

This **Learner's Handbook** is your practical guide for the hands-on exercises. Think of it less as a manual and more as a co-pilot: it will walk you through each step, explain what's happening and why, and help you build real, working test automation by the end of the session.

!!! info "Prerequisites"
    **Prerequisites:** Software Test Engineer Training - Learning Plan

    **Course Duration:** 1 day training + exercise submission

## What You'll Build

This training guides you through UiPath's Test Cloud, from planning and designing tests in Test Manager, to automating them in Studio, to running them at scale through Orchestrator — with agentic AI woven in at every step:

1. **Test Manager & Agentic Testing Fundamentals** — Get familiar with Test Manager, import a real project, and see how Agentic Testing (Autopilot for Testers and custom AI agents) fits into the testing lifecycle.
2. **Building Automated Tests in Studio** — Go from a manual test case to a fully automated one: Object Repository, data-driven testing, Orchestrator, publishing, and executing test sets.
3. **Building Your Own AI Agents** — Learn the anatomy of an agent (prompt, context, tools, escalations) and build one yourself.

## Learning Objectives

By the end of this course you will:

1. Get familiar with Test Manager capabilities and Agentic features
2. Work with testing artifacts: requirements, test cases, test sets, and test executions
3. Create automated test cases in low-code and coded approaches
4. Apply data-driven testing concepts in test automation by creating, managing, and utilizing test data
5. Publish and execute automated and manual test cases using Test Manager
6. Apply best practices in test automation development

Here's something worth holding onto as you work through this training:

Agentic testing doesn't replace testers — it extends them. Autopilot for Testers and custom AI agents handle the repetitive, pattern-based work so you can focus on judgment calls: what to test, why it matters, and how to interpret the results. Robots and agents work best together — robots bring determinism and control, agents bring adaptability and reasoning.

That's what this training is about: **building real, reliable test automation, augmented by AI at every step.**
